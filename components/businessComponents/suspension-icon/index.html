    # suspension-icon 组件

    该组件内部实现以 uview-plus 组件为基础，进行二次封装

    # 该组件主要应用场景 :

    • 自定义图标

    WARNING

    注意

    • 此组件内部使用 uview-plus uni-ui 组件为基础，详细说明请参 uview-plus 官方文档

    # 基础使用

    通过<g-suspension-icon>形式来调用, 具体使用方式如下:iconFlag 属性为 true 时，表示新增按钮使用的是默认的图标，iconFlag 属性为 false 时使用自己导入的图片路径;

    <!--默认新增 -->
<g-suspension-icon :iconFlag="true" @click="addHandle"></g-suspension-icon>
<!--自定义icon -->
<g-suspension-icon :iconFlag="true" :iconName="iconName" :iconColor="iconColor" :iconSize="iconSize" @click="addHandle"></g-suspension-icon>
<!--自定义图片 -->
<g-suspension-icon :iconFlag="false" @click="viewHandle"></g-suspension-icon>
<!--默认查看 -->
<g-suspension-icon :iconFlag="false" @click="viewHandle" :url="url" :background="background"  :fontSize="fontSize"  :color="coior"></g-suspension-icon>
<script lang="ts" setup>
	import { ref, onMounted } from 'vue'
	const iconName = ref('facebook-circle-fill')
	const iconColor = ref('#000')
	const iconSize = ref(200)

	const url = ref(''); // 创建响应式的 ref 来存储图片路径
	const background = ref('#ccc')
    const fontSize = ref('32rpx')
	const coior = ref('#000')
	onMounted(async () => {
        const importedImage = await import('@/assets/icons/check.png');
        url.value = importedImage.default;
		console.log('传入的url', url.value)
    })
	
	const addHandle = () => {
		console.log('新增')
	}
	const viewHandle = () => {
		console.log('查看')
	}
</script>

    参数 说明 类型 默认值 可选值
    iconFlag 是否是iocn Boolean true false
    iconName 图标名称 String plus-circle-fill -
    iconColor 图标颜色 String #02BC92 -
    iconSize 图标大小 String，Number 126rpx
    imgBoxWidth 图片盒子的宽 String 112rpx -
    imgBoxHeight 图片盒子的高 String 112rpx -
    imgBoxBackground 图片盒子的背景色 String - -
    url 自定义图片路径 String - -
    imgWidth 图片的宽 String 56rpx -
    imgHeight 图片的高 String 56rpx -
