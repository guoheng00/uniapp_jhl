# npm 安装

基于 uni-ui 与 uview-plus 进行封装

WARNING

温馨提示

  • 若是疑问,请参考 uni-ui 与 uview-plus 配置文档
  • node 18+ (Vue3/Vite 版要求 node 版本 18+、20+)
  • 创建命令 npx degit dcloudio/uni-preset-vue#vite-ts 你的项目名称
  • 第一步
// 安装sass
npm i sass -D

// 安装sass-loader，注意需要版本10，否则可能会导致vue与sass的兼容问题而报错
npm i sass-loader@10 -D

  • 第六步 此配置需要在项目 src 目录的 pages.json 中进行。
// pages.json
{
	"easycom": {
		"autoscan": true,
		// 注意一定要放在custom里，否则无效，https://ask.dcloud.net.cn/question/131175
		"custom": {
	    "^u--(.*)": "uview-plus/components/u-$1/u-$1.vue",
      "^up-(.*)": "uview-plus/components/u-$1/u-$1.vue",
      "^u-([^-].*)": "uview-plus/components/u-$1/u-$1.vue",
      "^uni-(.*)": "@dcloudio/uni-ui/lib/uni-$1/uni-$1.vue"
		}
	},

	// 此为本身已有的内容
	"pages": [
		// ......
	]
}
