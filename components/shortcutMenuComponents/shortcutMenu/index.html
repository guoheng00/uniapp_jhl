    # 菜单导航 组件

    # 该组件主要应用场景 :

    • 该组件是一个快捷导航的菜单。

    # 基础使用

    通过<g-shortcut-menu></g-shortcut-menu>形式来调用，通过:arrayList="arrayList"传入数组对象，对象包括name、image、id值，注意image必须是线上url地址。@selectItem可获得选中的值进行相应的操作。

          <g-shortcut-menu @selectItem="selectItem" :arrayList="arrayList"></g-shortcut-menu>
