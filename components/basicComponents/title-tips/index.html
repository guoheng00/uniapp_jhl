    # title-tips 组件

    该组件内部实现以 uview-plus 组件为基础，进行二次封装

    # 该组件主要应用场景 :

    • 警告提示

    WARNING

    注意

    • 此组件内部使用 uview-plus 组件为基础，详细说明请参 uview-plus 官方文档

    # 基础使用

    通过<g-title-tips>形式来调用，标题内容通过title传入

    <g-title-tips :title="title" ></g-title-tips>

<script lang="ts" setup>
	import { ref } from 'vue'
	const title = ref('中度警示文字')
</script>

    # 自定义样式

    <g-title-tips :title="title" :background="background" :iconName="iconName" :iconColor="iconColor" :iconSize="iconSize" :fontSize="fontSize"  :color="color"></g-title-tips>

<script lang="ts" setup>
  import { ref } from 'vue'
	const title = ref('中度警示文字')
	const background = ref('#ccc')
	const iconName = ref('grid-fill')
	const iconColor = ref('#000')
  const iconSize = ref('20px')
  const fontSize = ref('32rpx')
  const color = ref('#000')
</script>

    # 绑定事件

  import { ref } from 'vue'
	const titleThree = ref('点击提示')
    const handleClick = () => {
		console.log('点击了标题提示')
	}

    # Props

    参数 说明 类型 默认值 可选值
    title 显示的文字 String - -
    background 背景颜色 String #fff6c7 -
    iconName 图标名 String info-circle-fill -
    iconColor 图标颜色 String #fcc200 -
    iconSize 图标大小 String 64rpx -
    fontSize 标题的文字大小 String 26rpx -
    color 标题的文字颜色 String #fcc200 -
