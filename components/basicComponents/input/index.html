    # 输入类型

    • 通过 type 设置输入框的类型，默认 text
    • 通过 placeholder 设置输入框为空时的占位符
    • 通过 border 配置是否显示输入框的边框
    • 使用前置后置插槽 prefix suffix
    • 使用前后图标 prefixIcon suffixIcon
    • 可清空字符 clearable
    • 绑定@change 事件
    <g-input placeholder="四周边框" clearable border="surround" v-model="value" @change="change"></g-input>
<g-input placeholder="底部边框禁用" disabled border="bottom" v-model="value" @change="change"></g-input>
<g-input placeholder="圆角" clearable shape="circle" type="password" v-model="value" @change="change"></g-input>
<g-input placeholder="前置图标" prefixIcon='photo' v-model="value" @change="change"></g-input>
<g-input placeholder="后置图标" shape="circle" suffixIcon='photo' v-model="value" @change="change"></g-input>
<g-input placeholder="前置插槽" shape="circle" v-model="value" @change="change">
	<template #prefix>
		<text>前置插槽</text>
		</template>
	</g-input>
<g-input placeholder="后置插槽" shape="circle" v-model="value" @change="change">
	<template #suffix>
		<view @click="handleClick">后置插槽</view>
	</template>
</g-input>

    # 输入框类型

    WARNING

    注意

    根据 uni-app 的 input 组件文档，H5 和 APP 中只要声明了 password 参数(无论 true 还是 false)，type 均失效，此时为了防止 type=number 时，又存在 password 属性，type 无效，此时需要设置 password 为 undefined

    综述：输入框的类型可通过配置 type 来设置：

    • text-文本输入键盘。
    • number-数字输入键盘，app-vue 下可以输入浮点数，app-nvue 和小程序平台下只能输入整数。
    • idcard-身份证输入键盘，微信、支付宝、百度、QQ 小程序。
    • digit-带小数点的数字键盘，App 的 nvue 页面、微信、支付宝、百度、头条、QQ 小程序。
    • password-等同于设置 password 为 true 的效果

    # Api

    # Props

    参数 说明 类型 默认值 可选值
    value 输入的值 Number | String - -
    type 输入框类型，见上方说明 String text number | idcard | digit | password
    disabled 是否禁用输入框 Boolean false true
    disabledColor 禁用状态时的背景色 String #f5f7fa -
    clearable 是否显示清除控件 Boolean false true
    password 是否密码类型 Boolean false true
    maxlength 最大输入长度，设置为 -1 的时候不限制最大长度 String | Number -1 -
    placeholder 输入框为空时的占位符 String - -
    placeholderClass 指定 placeholder 的样式类 String input-placeholder /deep/ 类名前需加上字符串
    placeholderStyle 指定 placeholder 的样式 String | Object color: #c0c4cc -
    showWordLimit 是否显示输入字数统计 Boolean false true
    confirmType 设置右下角按钮的文字 String done send | search | next | go | done
    confirmHold 点击键盘右下角按钮时是否保持键盘不收起 Boolean false true
    holdKeyboard focus 时，点击页面的时候不收起键盘 Boolean false true
    focus 自动获取焦点 Boolean false true
    autoBlur 键盘收起时自动失去焦点 Boolean false true
    disableDefaultPadding 是否去掉 iOS 下的默认内边距 Boolean false true
    cursor 指定 focus 时光标的位置 String | Number -1 -
    cursorSpacing 输入框聚焦时底部与键盘的距离 String | Number 30 -
    selectionStart 光标起始位置，自动聚集时有效 String | Number -1 -
    selectionEnd 光标结束位置，自动聚集时有效 String | Number -1 -
    adjustPosition 键盘弹起时是否自动上推页面 Boolean true false
    inputAlign 输入框内容对齐方式 String left left | center | right
    fontSize 输入框字体的大小 String | Number 15px -
    color 输入框字体颜色 String #303133 -
    prefixIcon 输入框前置图标 String - -
    prefixIconStyle 前置图标样式 String | Object - -
    suffixIcon 输入框后置图标 String - -
    suffixIconStyle 后置图标样式 String | Object - -
    border 边框类型 String surround bottom | none
    readonly 是否只读 Boolean false true
    shape 输入框形状 String square circle
    formatter 输入过滤或格式化函数(如需兼容微信小程序，则只能通过 setFormatter 方法) Function null -
    customStyle 定义需要用到的外部样式 Object - -

    # Event

    事件名 说明 回调参数 版本要求
    handleBlur 输入框失去焦点时触发 value：内容值 -
    handleFocus 输入框聚焦时触发 - -
    handleConfirm 点击完成按钮时触发 value：内容值 -
    handleKeyboardheightchange 键盘高度发生变化时触发此事件 - 微信小程序 2.7.0+、App 3.1.0+
    handleInput 内容发生变化时触发此事件 value：内容值 -
    handleChange 内容发生变化时触发此事件 value：内容值 -
    handleClear 点击清空内容时触发 - -
