    # Search 搜索框组件

    该组件内部实现以 uview-plus 组件为基础，进行二次封装

    # 该组件主要应用场景 :

    • 数据搜索功能

    WARNING

    注意

    • 此组件内部使用 uview-plus 组件为基础，详细说明请参 uview-plus 官方文档

    # 基础使用

    通过<g-search>形式来调用，

    <g-search v-model="keyword" :placeholder="searchPlaceholder"></g-search>
<g-search v-model="keyword1" :placeholder="searchPlaceholder" @handleSearch="handleSearch" @handleClear="handleClear"></g-search>
<script lang="ts" setup>
import { ref } from 'vue';
	// 响应式数据
	const keyword = ref<string>('');
	const keyword1 = ref<string>('已输入样式');
	const searchPlaceholder = "在输入框中输入搜索"
	// 点击搜索 回车 进行搜索
	const handleSearch = (value : string) => {
		console.log(value);
	}
	// 点击清空
	const handleClear = (value : string) => {
		console.log(value,'----');
	}
</script>

    # Api

    # Props

    参数 说明 类型 默认值 可选值
    value 数据绑定
    placeholder 输入框内提示信息 String

    # Event

    事件名 说明 回调参数
    handleSearch 点击搜索 回车触发 value：当前值
    handleClear 点击清空触发(必须在进行一次搜索后才会触发) value：当前值
