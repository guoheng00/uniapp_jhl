# icon 组件

基于字体的图标集，包含公司所用场景的图标(建立公司 icon 库)。

# 该组件主要应用场景 :

WARNING

注意

因为在 nvue 下暂时无法解决自定义图标的问题，所以此功能暂缓推出, 但是可以使用图片图标来进行自定义.

# 基础使用

通过<jhl-icon>形式来调用，设置 name 参数为图标名即可。其中 color 默认为#606266，size 默认为 16px

<jhl-icon name='photo'></jhl-icon>
1

# 修改图标样式

  • 通过 color 参数修改图标的颜色
  • 通过 size 参数修改图标的大小，单位为 px(是 fontsize)
// 28 * 28
<jhl-icon name='photo' color='#2979ff' size='28'></jhl-icon>
1
2

# 图片图标

  • 可根据 width height 来控制尺寸
// 28 * 28
<jhl-icon
  name='https://cdn.uviewui.com/uview/example/button.png'
  label='图片图标'
  width='16'
  height='16'
></jhl-icon>

# 绑定事件

<text class="label">点击图标</text>
<jhl-icon name="https://cdn.uviewui.com/uview/example/button.png" width='30' height='30' index='1'
				@click="handleClick"></jhl-icon>
<jhl-icon name="https://cdn.uviewui.com/uview/example/button.png" width='30' height='30' index='2'
				@click="handleClick"></jhl-icon>
<script setup>
	import {
		toast
	} from '../../uni_modules/jhl-ui/libs/function';
	const handleClick = (index) => {
		toast(`当前index : ${index}`)
	}
</script>

# Api

# Props

参数 说明 类型 默认值 可选值
name 图标名称，见示例图标集，如名称带有/，会被认为是图片图标 String - -
color 图标颜色 String color['u-content-color'] -
size 图标字体大小，单位默认 px String | Number 16px -
bold 是否显示粗体 Boolean false -
index 一个用于区分多个图标的值，点击图标时通过 click 事件传出 String | Number - -
hoverClass 图标按下去的样式类，用法同 uni 的 view 组件的 hover-class 参数 String - -
customPrefix（暂不可用） 自定义字体图标库时，请使用图片图标 String uicon -
label 图标右侧/下方的 label 文字 String | Number - -
labelPos label 相对于图标的位置 String right bottom / top / left
labelSize label 字体大小，单位默认 px String | Number 15px -
labelColor label 字体颜色 String color['u-content-color'] -
space label 与图标的距离，单位默认 px String | Number 3px -
imgMode 图片裁剪、缩放的模式，image 组件原生属性，详见：image String - -
width name 为图片路径时图片的宽度，单位默认 px String | Number - -
height name 为图片路径时图片的高度，单位默认 px String | Number - -
top 图标到顶部的距离，如果某些场景，如果图标没有垂直居中，可以调整此参数，单位默认 px String | Number 0 -
stop 是否阻止事件传播 Boolean false -

# Event

事件名称 说明 回调参数
click 点击事件,点击时触发 index: 通过 props 传递的 index 值
