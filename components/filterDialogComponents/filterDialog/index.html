    # 弹出层筛选 组件

    # 该组件主要应用场景 :

    • 该组件主要应用于弹出层选项筛选。

    # 基础使用

    通过<g-filter-dialog></g-filter-dialog>形式来调用，通过:show="show"传入控制弹出层显示隐藏，通过:arrayList="arrayList"传入选项标题和item选项。

              <g-filter-dialog :show="show" :arrayList="arrayList" @itemChange="itemChange" @close="close" @confirm="confirm"
      @resetBtn="resetBtn"></g-filter-dialog>
      let arrayList = ref([
        {
          itemTitle: '交易类型',
          itemList: [
            { name: '全部', value: 'ALL'},
            { name: '收款', value: 'OWN' },
            { name: '退款', value: 'CHANNEL' },
          ],
          scopeSelect: 'ALL'
        },
        {
          itemTitle: '支付方式',
          itemList: [
            { name: '全部', value: 'ALL'},
            { name: '微信支付', value: 'WECHAT' },
            { name: '支付宝', value: 'ALIPAY' },
          ],
          scopeSelect: 'ALL'
        }
      ])

    注意

    • 此组件传入arrayList数组中itemTitle是选项标题，scopeSelect是默认值，默认选中全部。可根据需要修改默认值。
    • itemChange事件是获取选中的值，其中resetBtn是重置，重置时需要传入初始值即可。
