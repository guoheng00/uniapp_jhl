    # TreeSelect 分类选择 组件

    # 该组件主要应用场景 :

    • 用于从一组相关联的数据集合中进行选择。

    # 基础使用

    通过<g-tree-select :show="show" @sureSelect="sureSelect" :businessOpts="businessOpts" :businessCategoryIdx="businessCategoryIdx" :businessChildrenCategoryIdx="businessChildrenCategoryIdx" @closeBtn=closeBtn @operateFirst="operateFirst" @operateLast="operateLast"></g-tree-select>形式来调用，通过:businessOpts="businessOpts"传入数组对象，若用二级选择，即只选择经营类别，传入:show="show"，@sureSelect="sureSelect"可获取选择项的值。若选项三级省市区，传入:areaShow="areaShow" ，@areaSureSelect="areaSureSelect"可获取选中项的值。

    WARNING

    注意

    • 传入的列表选择是完整的数组对象，包含子集下面的children数组，以name,id形式命名传入。
    • 经营类别支持编辑回显，若需使用编辑回显需传入businessCategoryIdx参数，即一级分类选中索引值，传入businessChildrenCategoryIdx即二级分类选中索引值。其中operateFirst方法接收一级菜单选中索引值，即回传回去，避免页面直接修改组件值，operateLast方法接收二级菜单选中索引值，即回传回去，避免页面直接修改组件值。
    • 省市区选择支持编辑回显，若需使用编辑回显需传入proviceCategoryIdx参数，即省级选中索引值，传入cityCategoryIdx参数即市级选中索引值，传入areaCategoryIdx参数即区级选中索引值，其中 @selectProviceValue函数接收选中省级索引并回传回去，@selectCityValue函数接收选中市级索引参数并回传回去，@selectAreaValue函数接收区级选中索引参数并回传回去，避免页面直接修改组件值。
